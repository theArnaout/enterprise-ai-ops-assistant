# Requirements Backlog

## User Stories

### US-1
As an analyst, I want to explore operational data using SQL so that I can validate data quality and consistency.

**Acceptance Criteria**
- Queries identify missing or invalid values
- Results are repeatable and well-documented

---

### US-2
As a reviewer, I want to view operational trends and exceptions in a dashboard so that I can focus attention where it is needed most.

**Acceptance Criteria**
- Dashboard metrics match SQL outputs
- Visuals are clear and easy to interpret

---

### US-3
As a user, I want an AI assistant to explain notable patterns or anomalies in the data so that I can understand issues more quickly.

**Acceptance Criteria**
- AI responses reference the underlying data
- Outputs are easy to edit or refine

---

### US-4
As a team member, I want AI-assisted draft summaries so that documentation is faster and more consistent.

**Acceptance Criteria**
- Drafts follow a simple, structured format
- Human review is required before use
