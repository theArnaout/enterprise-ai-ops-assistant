# Risks and Constraints

## Identified Risks
- Misinterpretation of AI-generated explanations
- Overreliance on automated summaries
- Incomplete or inconsistent data

## Mitigation Strategies
- Require human review for all AI outputs
- Clearly document what the AI assistant can and cannot do
- Validate data using SQL before analysis or reporting

## Constraints
- No automated decisions or approvals
- No production system integrations
- Dataset represents simulated or non-sensitive data
